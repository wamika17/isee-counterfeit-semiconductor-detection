# Electrical Testing

Electrical verification is used to determine whether the IC behaves according to its expected specifications.

## Measurements

- Logic Outputs
- Input Voltage
- Output Voltage

## Verification Process

1. Apply all logic input combinations.
2. Read output logic.
3. Measure electrical characteristics.
4. Compare measured values with expected values.
5. Generate numerical features for machine learning.

## Output Features

- Truth Table Accuracy
- Average Input Voltage
- Average Output Voltage
- Voltage Drop
- Failed Logic Tests
