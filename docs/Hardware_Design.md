# Hardware Design

## Controller

Arduino Mega 2560

## Device Under Test

7400 NAND Gate IC

## Supporting Components

- LM324 Operational Amplifier
- Breadboard
- USB Communication

## Responsibilities

The Arduino generates logic input combinations, acquires analog voltage measurements, and transmits the collected data to the host computer for processing.

## Pin Mapping

| Arduino Pin | Function |
|--------------|----------|
| D2 | Input A |
| D3 | Input B |
| D4 | Output Y |
| A0 | Input Voltage |
| A1 | Output Voltage |
