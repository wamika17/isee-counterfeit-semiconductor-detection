# Electrical Analysis

This module performs electrical verification of semiconductor ICs by comparing their measured electrical behavior against the expected characteristics of genuine devices.

Unlike visual inspection alone, electrical testing evaluates the functional performance of an integrated circuit, making it possible to detect recycled, damaged, or counterfeit components that may appear visually authentic.

---

## Objectives

- Verify logic gate functionality.
- Measure input and output voltages.
- Compare measured values against expected thresholds.
- Extract numerical features for machine learning classification.
